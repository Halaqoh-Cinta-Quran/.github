
# Halaqoh Cinta Qur'an


![Logo](https://qaulunmarufun.com/wp-content/uploads/2019/08/logo-HCQ1.png)



Adalah sebuah lembaga akademik yang berfokus pada pengajaran Tahsin dan Tahfidz dengan Riwayat (sanad) Imam Hafs dari Ashim dengan Thariq Syatubiyyah.

Halaqoh Cinta Qur'an berada di bawah Yayasan Qaulun Ma'rufun, yang berbasis di Tangerang Selatan, Banten.
